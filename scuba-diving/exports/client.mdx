### **openMenu**

Opens the main scuba diving menu.

```lua
exports.lation_diving:openMenu()
```

### isGearEquipped

Returns `true` or `false` if scuba gear is currently equipped.

```lua
exports.lation_diving:isGearEquipped()
```
